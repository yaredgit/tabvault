Data Collection

TabVault does not collect, transmit, or store any personal data. All data (saved tabs) is stored locally in your browser using Chrome's storage API and is automatically removed when tabs are restored.

Data Storage

The only data stored locally includes:

  • Tab URLs
  • Tab titles
  • Timestamps (for organization)

This data is stored only on your device and is automatically removed when you restore tabs.

Third-Party Services

TabVault does not use any third-party services, analytics, or tracking.
